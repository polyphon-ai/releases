# Polyphon Releases

This repository is used to publish releases of [Polyphon](https://polyphon.ai) and serves as the central place for community feedback and discussion.

## Installation

Release artifacts are attached to each [GitHub Release](../../releases). Download the appropriate binary or package for your platform.

## Versioning

Polyphon follows [Semantic Versioning](https://semver.org/). Release notes are included with each release.

## Feedback

- **Bug reports** — [Open a bug report](../../issues/new?template=bug_report.yml)
- **Feature requests** — [Open a feature request](../../issues/new?template=feature_request.yml)
- **Questions & discussion** — [Start or join a discussion](../../discussions)

## Security

Please do not report security vulnerabilities through public issues. See [SECURITY.md](SECURITY.md) for how to disclose a vulnerability responsibly.
